# Installation

This guide provides step-by-step instructions to install and set up your project, either by selecting the **Starter Kit** or the **Full Kit** version.

---

## Prerequisites

Before you begin, make sure you have the following installed on your system:

1. **Node.js** (version 18.18 or later)
   - Download it from: [https://nodejs.org/](https://nodejs.org/).
2. **npm** (Node Package Manager)  
   npm comes bundled with Node.js. You can verify if it's installed by running:

   ```bash
   npm --version
   ```

3. **Supported Operating Systems**:

   - **macOS**
   - **Windows** (including WSL)
   - **Linux**

4. **Code Editor**  
   A code editor is required to edit your project files. We recommend [VSCode](https://code.visualstudio.com/) but feel free to use your preferred editor.

---

## Download the Project

Follow these steps to download and choose the version that fits your needs:

1. **Download** the zipped project file and save it to your desired directory.
2. **Extract** the zip file:
   - Inside, you'll find two folders: `typescript-version` and `javascript-version`.
   - Choose a version based on your preferred setup:
     - **Full Kit**: Includes all features, apps, and pages for a comprehensive project.
     - **Starter Kit**: A minimal setup containing core features for easy customization.
3. Place your selected version in your workspace directory and open it in your preferred code editor.

---

## Dependency Installation

After extracting the project and choosing the version, install the required dependencies using npm:

```bash
npm install
```

---

## Launch Your Project

Start the development server using the following command:

```bash
npm run dev
```

The project will launch at **http://localhost:3000** by default.
