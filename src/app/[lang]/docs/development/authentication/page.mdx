# Authentication

This guide provides step-by-step instructions to **add** authentication in your application using **NextAuth.js**.

---

## Adding Authentication

### Install Authentication Dependencies

Install the necessary packages to set up authentication with **NextAuth.js**:

```bash
npm install next-auth @auth/prisma-adapter @prisma/client prisma
```

---

### Configure Environment Variables

Add the following environment variables to your `.env` file. Refer to the **full-kit** for sample values:

```env
API_URL=
NEXTAUTH_URL=
NEXTAUTH_SECRET=
```

---

### Add Required Files

Copy the following files and directories from the **full-kit**:

- **Prisma Schema**: `prisma/schema.prisma`
- **API Route for NextAuth.js**: `src/app/api/auth/[...nextauth]/route.ts`
- **NextAuth Configuration**: `src/configs/next-auth.ts`
- **Prisma Client**: `src/lib/prisma.ts`
- **Authentication Layout**: `src/app/(blank-layout)/(auth)`
- **NextAuth Provider**: `src/providers/next-auth-provider.tsx`

---

### Update the `Providers` Component

Ensure the `next-auth-provider` is included in your project by updating the `src/providers/index.tsx` file. Refer to the **full-kit** for an example implementation.

---

### Update the `package.json` Scripts

Update the `scripts` section in your `package.json` file to include the following:

```json
"scripts": {
    ...
    "migrate": "npx prisma migrate dev",
    "postinstall": "prisma generate"
}
```

---

### Set Up Prisma

Run the following commands to set up Prisma and apply migrations:

```bash
npm run migrate
npx prisma generate
```

---
